Аргументы программы:

1) Путь к папке с изначальными изображениями (должна существовать)
2) Путь к папке для выходных изображений (может не существовать)
3) Размер одного (и соответственно каждого) выходного изображения

#############################################################################################################################################################

Запуск программы:

1) клик по ярлыку CutFaces
2) запуск run_app.bat
3) а) запустить cmd
   б) перейти в папку с программой
   в) вызвать строку CutFaces.exe <аргументы программы через пробел>
